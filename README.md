# Amazon Sales Dashboard (Excel)

## Project Overview
This project is an interactive Amazon Sales Dashboard created in Microsoft Excel to analyze sales performance, delivery efficiency, and customer purchasing trends.

## Objectives
- Analyze sales across different regions
- Identify top-selling products
- Track delivery performance
- Monitor payment methods and order trends

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

## Key Insights
- North Region generated the highest sales.
- Smartphones were the best-selling product category.
- 77% of orders were delivered through fast delivery.
- Online payments were the most preferred payment method.

## Dashboard Features
- Region Filter
- Month Filter
- Delivery Status Filter
- Payment Method Filter
- Interactive Charts and KPIs
